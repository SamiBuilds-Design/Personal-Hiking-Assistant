# Setup Guide

This guide explains how to import and configure the Personal Hiking Assistant workflow in n8n.

## Prerequisites

- n8n account
- OpenAI API Key
- Google Sheets credentials
- Gmail credentials
- Google Calendar credentials

## Setup Steps

1. Import `workflow.json` into n8n.
2. Configure all required credentials.
3. Connect your Google services.
4. Add your OpenAI API key.
5. Activate the workflow.
6. Test the workflow.

## Workflow Components

- AI Agent (OpenAI)
- Weather Tool
- Google Sheets
- Gmail
- Google Calendar
- Memory
